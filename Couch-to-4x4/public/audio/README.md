# Audio Cues

These cue files are MP3 audio assets used by the Couch to 4x4 app.

## Available Cue Files

- `warmup_start.mp3` - Warm-up phase start cue
- `work_start_coached.mp3` - Work interval start cue with coaching
- `rest_start_coached.mp3` - Rest interval start cue with coaching
- `warning_10.mp3` - 10-second warning before phase transition
- `warning_redline.mp3` - Final 3-second redline warning
- `work_halfway.mp3` - Halfway point marker for work intervals
- `cooldown_start.mp3` - Cool-down phase start cue
- `workout_complete.mp3` - Workout completion celebration

## Notes

These files are referenced by their paths (e.g., `/audio/warmup_start.mp3`) in the application code.
